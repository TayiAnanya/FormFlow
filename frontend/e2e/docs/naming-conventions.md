# Naming conventions

See [`specs/playwright/01-IMPLEMENTATION-SPEC.md`](../../../specs/playwright/01-IMPLEMENTATION-SPEC.md) §6.2.

| Pattern | Example |
|---|---|
| Page | `login.page.ts` |
| Component | `dynamic-form.ts` |
| Workflow | `auth.workflow.ts` |
| Spec | `auth.flows.spec.ts` |
| Fixture | `authentication.fixture.ts` |
| Adapter | `local-storage.adapter.ts` |
| Data pack | `happy.packs.ts` |

Tags: `shared/config/test-tags.ts` (`TAGS.smoke`, `TAGS.critical`, …).
